        Guia de utilização Newton-Raphson 
*********************************
        Formas de entrada:

            seno(x) = sin(x)
            cosseno(x) = cos(x)
            tangente(x) = tan(x)
            ln(x) = log(x)
            constante e = e
            π = pi

        elevações:
            x ao quadrado = x**2
            x ao cubo = x**3
            x a n potência = x**n

        raíz quadrada de x:
            sqrt(x)
        raíz cúbica de x:
            x**(1/3)
        raíz n de x:
            x**(1/n)

        ***********************
        funções modo de escrita
        exemplos:
            2*x+x**2+1

            sin(x)-x**2

            log(x)+5*x**3

            sqrt(x)-2*x

            x**3+2*pi
